# Laravel-Vue-App

## Run Locally

Clone the project

```bash
  git clone https://github.com/jaysontolentino/laravel-vue-spa.git
```

Go to the project directory

```bash
  cd laravel-vue-spa
```

Install dependencies

```bash
  yarn
```

or

```bash
  npm install
```

Start the app

```bash
  yarn dev
```

or

```bash
  npm run dev
```
